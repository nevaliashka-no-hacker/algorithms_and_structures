# Задачи для самостоятельной работы по семинару "Очередь"


1. Реализовать очередь на двух стеках.
2. Написать функцию для проверки корректности скобок с использованием очереди (по аналогии с тем что делали со стеком на лабах).
3. Решить 3 задачи из списка LeetCode на выбор.
4. Реализовать систему обработки заказов в ресторане (с приоритетами).
