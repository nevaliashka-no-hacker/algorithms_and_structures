# Задачи для самостоятельной работы по семинару "Скользящее окно и смежные техники"

Вариант берётся по последней цифре вашего номера в списке группы. Для кратных 10 берётся 10 вариант.

Например:
- Номер 3 → Вариант 3
- Номер 17 → Вариант 7
- Номер 20 → Вариант 10
- Номер 25 → Вариант 5

> [!TIP]
> Перед выполнением стоит заглянуть в [шаблон](/sem_2/template.md), в котором есть заготовки функций.

## Вариант 1

**Задача 1.1.** Дан массив целых чисел `nums` и число `k`. Найти максимальную сумму среди всех подмассивов длины `k`.

```
Вход:  nums = [2, 1, 5, 1, 3, 2], k = 3
Выход: 9
Пояснение: подмассив [5, 1, 3] имеет сумму 9
```

**Задача 1.2.** Дан массив положительных целых чисел `nums` и число `target`. Найти минимальную длину непрерывного подмассива, сумма которого больше или равна `target`. Если такого подмассива нет, вернуть 0.

```
Вход:  nums = [2, 3, 1, 2, 4, 3], target = 7
Выход: 2
Пояснение: подмассив [4, 3] имеет сумму 7 и длину 2
```

**Задача 1.3.** Дан отсортированный по возрастанию массив целых чисел `nums` и число `target`. Найти два элемента, сумма которых равна `target`. Вернуть их индексы (1-based). Гарантируется, что решение единственно.

```
Вход:  nums = [2, 7, 11, 15], target = 9
Выход: [1, 2]
```

## Вариант 2

**Задача 2.1.** Дана строка `s` и число `k`. Найти максимальное количество гласных букв (a, e, i, o, u) среди всех подстрок длины `k`.

```
Вход:  s = "abciiidef", k = 3
Выход: 3
Пояснение: подстрока "iii" содержит 3 гласных
```

```
Вход:  s = "abcabcbb"
Выход: 3
Пояснение: "abc"
```

**Задача 2.3.** Дан массив целых чисел `nums` и число `k`. Подсчитать количество непрерывных подмассивов, сумма которых равна `k`.

```
Вход:  nums = [1, 1, 1], k = 2
Выход: 2
Пояснение: [1,1] начиная с индекса 0 и [1,1] начиная с индекса 1
```

## Вариант 3

**Задача 3.1.** Дан массив целых чисел `nums` и число `k`. Определить, существуют ли в массиве два одинаковых элемента, расстояние между индексами которых не превышает `k`.

```
Вход:  nums = [1, 2, 3, 1], k = 3
Выход: True
```

```
Вход:  nums = [1, 2, 3, 1, 2, 3], k = 2
Выход: False
```

**Задача 3.2.** Дан бинарный массив `nums` (содержит только 0 и 1) и число `k`. Можно заменить не более `k` нулей на единицы. Найти длину наибольшего непрерывного подмассива, состоящего из единиц.

```
Вход:  nums = [1, 1, 0, 0, 1, 1, 1, 0, 1], k = 2
Выход: 8
Пояснение: заменяем нули на позициях 2 и 7
```

**Задача 3.3.** Дан массив целых чисел `nums`. Найти все уникальные тройки элементов, сумма которых равна нулю. Тройки не должны повторяться.

```
Вход:  nums = [-1, 0, 1, 2, -1, -4]
Выход: [[-1, -1, 2], [-1, 0, 1]]
```

```

```
