📊 Сортировки

Что такое сортировка?

Сортировка — это алгоритм упорядочивания элементов в определённом порядке (возрастание, убывание, лексикографический). Фундаментальная операция в computer science.

Ключевые понятия:
  • Стабильность — равные элементы сохраняют исходный порядок
  • In-place — алгоритм использует O(1) дополнительной памяти
  • Адаптивность — ускоряется на частично отсортированных данных

Классификация алгоритмов сортировки

По сложности времени
O(n²): Bubble, Insertion, Selection
O(n log n): Merge, Quick, Heap
O(n): Counting, Radix, Bucket (специальные)
По методу
Сравнениями: Quick, Merge, Heap
Подсчётом: Counting, Radix
Вставками: Insertion, Shell

Встроенная сортировка в языках

ЯзыкФункцияАлгоритмСложность
Pythonlist.sort(), sorted()Timsort (гибрид Merge + Insertion)O(n log n), O(n) на отсорт.
Cqsort()Quick Sort (обычно)O(n log n) сред., O(n²) худ.
C++std::sort()Introsort (Quick + Heap + Insertion)O(n log n) гарант.
⚠️ Для собеседования: Не говорите «я просто вызову .sort()». Нужно знать, как реализовать вручную.

Где используются сортировки?

  • 🗄️ Базы данных: ORDER BY, индексы
  • 🔍 Поиск: Бинарный поиск требует отсортированных данных
  • 📈 Визуализация: Графики, рейтинги
  • 🖼️ Графика: Z-буфер, определение видимости
  • 🤖 ML: Подготовка данных, kNN
