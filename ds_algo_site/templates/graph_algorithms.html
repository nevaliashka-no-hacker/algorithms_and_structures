⚙️ Алгоритмы графов (18)

🎯 Графы — 30% экзаменационных билетов. Учите BFS, DFS, Dijkstra в первую очередь!

📌 Обходы

1. DFS (поиск в глубину) ⭐⭐⭐⭐⭐

Рекурсивный обход: идём по первому соседу до тупика, возвращаемся (backtracking).

ВремяO(V + E)ПамятьO(V) стек

Применение: поиск циклов, компоненты связности, топологическая сортировка.

2. BFS (поиск в ширину) ⭐⭐⭐⭐⭐

Обход по уровням с очередью.

ВремяO(V + E)ПамятьO(V)

Применение: кратчайший путь в невзвешенном графе, поиск в ширину.

📌 Кратчайшие пути

3. Алгоритм Дейкстры ⭐⭐⭐⭐⭐

От одной вершины до всех в взвешенном графе с неотрицательными весами. Приоритетная очередь.

ВремяO((V+E) log V) с heapПамятьO(V)
4. Алгоритм Беллмана-Форда ⭐⭐⭐

Работает с отрицательными весами. V-1 итераций релаксации всех рёбер. Обнаруживает отрицательные циклы.

ВремяO(V * E)ПамятьO(V)
5. Алгоритм Флойда-Уоршелла ⭐⭐⭐

Все пары кратчайших путей. Динамическое программирование.

ВремяO(V³)ПамятьO(V²)
6. A* (A-star) ⭐⭐

Дейкстра + эвристика (расстояние до цели). Поиск пути в играх, навигация.

📌 Минимальное остовное дерево (MST)

7. Алгоритм Крускала ⭐⭐⭐⭐

Сортируем рёбра по весу, добавляем если не создаёт цикл (DSU).

ВремяO(E log E)ПамятьO(V)
8. Алгоритм Прима ⭐⭐⭐⭐

Начинаем с одной вершины, добавляем ближайшую не в дереве. Приоритетная очередь.

ВремяO((V+E) log V)ПамятьO(V)

📌 DAG и топологическая сортировка

9. Алгоритм Кана (BFS) ⭐⭐⭐⭐⭐

Считаем входящие степени, добавляем вершины с 0 в очередь.

ВремяO(V + E)ПамятьO(V)
10. Топологическая сортировка через DFS

DFS + стек. Добавляем вершину в стек после обработки всех потомков.

11. Кратчайший путь в DAG ⭐⭐⭐

Топологическая сортировка + релаксация рёбер в этом порядке. O(V+E).

📌 Связность, циклы, компоненты

12. Компоненты связности (DFS/BFS/Union-Find) ⭐⭐⭐⭐

Запускаем DFS/BFS из непосещённой вершины — все достигнутые в одной компоненте.

13. Поиск цикла (DFS с цветами) ⭐⭐⭐⭐

Три цвета: белый (не посещён), серый (в стеке), чёрный (обработан). Если встретили серый → цикл.

14. Двудольность графа (BFS с раскраской) ⭐⭐⭐

BFS, красим соседей в противоположный цвет. Если конфликт → не двудольный.

15. Мосты и точки сочленения (Тарьян) ⭐⭐⭐

DFS с временами входа и low-link значениями. Мост — ребро, удаление которого разбивает граф.

16. Сильные компоненты связности (Косарайю / Тарьян) ⭐⭐⭐

В ориентированном графе: компонента, где из любой вершины можно дойти до любой.

📌 Структуры данных для графов

17. DSU / Union-Find (Система непересекающихся множеств) ⭐⭐⭐⭐⭐

Две операции: find (сжатие пути) и union (по рангу/размеру). Почти O(1) амортизированное.

ВремяO(α(n)) — обратная функция Аккермана
18. Алгоритм иерархической кластеризации

Построение MST → удаление k-1 самых дорогих рёбер → k кластеров.
