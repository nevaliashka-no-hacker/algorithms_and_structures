🧩 Динамическое программирование (14 паттернов)

🎯 DP — самая сложная тема. Решается всё шаблонами. Выучите паттерны!

Что такое DP?

Метод решения задач путём разбиения на перекрывающиеся подзадачи с оптимальной подструктурой.

Два подхода:
  • Мемоизация (Top-Down): Рекурсия + кеш результатов
  • Табуляция (Bottom-Up): Заполнение таблицы от малых подзадач к большим
Когда применять DP:
  • Задача просит оптимум (минимум, максимум, количество способов)
  • Решение можно выразить через решения меньших подзадач
  • Есть перекрывающиеся вычисления (одни и те же подзадачи много раз)

📌 Паттерны DP

1. Числа Фибоначчи (введение) ⭐⭐⭐⭐⭐

F(n) = F(n-1) + F(n-2). Классика для объяснения мемоизации vs табуляции.

Без DPO(2ⁿ)С DPO(n)
2. Knapsack 0/1 (Рюкзак) ⭐⭐⭐⭐⭐

Даны веса и стоимости. Выбрать предметы с макс. стоимостью, умещаясь в вес W. Нельзя дробить.

ВремяO(n*W)ПамятьO(W) с оптимизацией
3. LCS (Longest Common Subsequence) ⭐⭐⭐⭐

Наибольшая общая подпоследовательность двух строк.

ВремяO(n*m)ПамятьO(n*m)
4. LIS (Longest Increasing Subsequence) ⭐⭐⭐⭐

DP за O(n²) или бинарный поиск за O(n log n).

5. Coin Change (Размен монет) ⭐⭐⭐⭐

Два варианта: минимальное число монет (BFS/DP) и количество способов (как рюкзак).

6. Редакционное расстояние (Левенштейна) ⭐⭐⭐

Минимальное число вставок/удалений/замен для превращения строки A в B. O(n*m).

7. DP по матрице (пути) ⭐⭐⭐

Путь с максимальной/минимальной суммой, количество путей. dp[i][j] = max(dp[i-1][j], dp[i][j-1]) + grid[i][j].

8. DP по подстрокам (палиндромы) ⭐⭐⭐

LPS (Longest Palindromic Substring), подсчёт палиндромных подстрок. dp[i][j] — палиндром ли s[i..j].

9. DP с битовыми масками ⭐⭐⭐

Когда n маленькое (≤ 20). Задача коммивояжёра (TSP): dp[mask][i] — посетили города mask, стоим в i.

10. DP на деревьях ⭐⭐⭐

Диаметр дерева, сумма путей, независимое множество на дереве. Рекурсия с возвратом двух состояний.

11. DP с префиксными суммами ⭐⭐

Оптимизация DP через префиксные суммы для сокращения размерности.

12. Partition Problem (разбиение) ⭐⭐⭐

Можно ли разбить массив на две части с равной суммой? Частный случай рюкзака.

13. DP по отрезкам ⭐⭐

Умножение цепочки матриц, burst balloons. dp[i][j] по всем k между i и j.

14. Мемоизация в графах ⭐⭐⭐

Подсчёт путей в DAG, наибольший путь. DP + топологическая сортировка.
