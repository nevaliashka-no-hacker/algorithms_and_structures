⚙️ Алгоритмы сортировок

🎯 Сортировки — основа алгоритмов! Знайте минимум Quick Sort, Merge Sort, Heap Sort и их Big O.

📌 Квадратичные сортировки O(n²)

1. Bubble Sort (Пузырьковая) ⭐

Попарное сравнение соседних элементов, обмен если порядок нарушен. Самый большой «всплывает» в конец за проход.

ВремяO(n²) худ/сред, O(n) лучПамятьO(1)Стабильность✅ Да

Применение: Учебный алгоритм, в реальном коде почти не используется.

2. Selection Sort (Сортировка выбором)

На каждом шаге находит минимум в неотсортированной части, меняет с первым неотсортированным.

ВремяO(n²) всегдаПамятьO(1)Стабильность❌ Нет
3. Insertion Sort (Сортировка вставками) ⭐⭐

Берёт элемент и вставляет в правильное место в отсортированной части. Эффективен на маленьких и почти отсортированных.

ВремяO(n²) худ/сред, O(n) лучПамятьO(1)Стабильность✅ Да

Используется: Внутри Timsort, для маленьких массивов (n < 10-50).

📌 Эффективные сортировки O(n log n)

4. Merge Sort (Сортировка слиянием) ⭐⭐⭐

Разделяй и властвуй: делим пополам → рекурсивно сортируем → сливаем отсортированные половины. Гарантированный O(n log n).

ВремяO(n log n) всегдаПамятьO(n)Стабильность✅ Да

Применение: Сортировка связных списков, внешняя сортировка.

5. Quick Sort (Быстрая сортировка) ⭐⭐⭐

Выбрать pivot → разделить на «меньше» и «больше» → рекурсивно сортировать части. Самый популярный на собеседованиях!

ВремяO(n log n) сред, O(n²) худПамятьO(log n) стекСтабильность❌ Нет (обычно)

Ключевые моменты: Выбор pivot (медиана трёх), partition (схема Lomuto или Hoare).

6. Heap Sort (Пирамидальная сортировка) ⭐⭐

Построить max-heap (heapify) → извлекать максимум в конец массива → восстанавливать кучу.

ВремяO(n log n) гарант.ПамятьO(1)Стабильность❌ Нет

Преимущество: O(1) памяти и гарантированный O(n log n).

7. Timsort (Python, Java, Android) ⭐⭐

Гибрид: находит упорядоченные подмассивы (run), сортирует их Insertion, сливает Merge. Адаптивный, стабильный.

ВремяO(n log n) худ, O(n) лучПамятьO(n)Стабильность✅ Да

📌 Линейные сортировки O(n) (не сравнениями)

8. Counting Sort (Сортировка подсчётом) ⭐⭐

Подсчитать частоту каждого значения, вычислить позиции. Только для целых чисел в известном диапазоне.

ВремяO(n + k), k — диапазонПамятьO(k)Стабильность✅ Да
9. Radix Sort (Поразрядная сортировка) ⭐⭐

Сортировка по разрядам (единицы → десятки → сотни). Использует стабильную сортировку (обычно Counting) для каждого разряда.

ВремяO(d * (n + b)), d — число разрядов, b — основаниеПамятьO(n + b)
10. Bucket Sort (Блочная/карманная сортировка)

Распределить элементы по buckets (диапазонам) → отсортировать каждый bucket → объединить.

ВремяO(n²) худ, O(n) сред при равномерном распределенииПамятьO(n)

📌 Другие алгоритмы сортировки

11. Shell Sort (Сортировка Шелла)

Модификация Insertion Sort: сравнивает элементы на расстоянии gap, постепенно уменьшая gap до 1.

ВремяO(n²) худ, O(n log n) сред с правильными gapПамятьO(1)
12. Comb Sort

Улучшенный Bubble Sort с уменьшающимся gap (обычно / 1.3). Устраняет проблему «черепах».

13. Intro Sort (Introsort) — C++ std::sort

Начинает с Quick Sort → если рекурсия глубокая, переключается на Heap Sort → для маленьких Insertion Sort. Гарантирует O(n log n).
