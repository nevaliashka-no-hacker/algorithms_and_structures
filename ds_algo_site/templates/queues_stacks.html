📦 Очереди и Стек

🥞 Стек (Stack) — LIFO

Last In, First Out — последний вошёл, первый вышел. Как стопка тарелок.

Основные операции:
  • push(x) — добавить на вершину O(1)
  • pop() — удалить с вершины O(1)
  • peek() / top() — посмотреть вершину без удаления O(1)
  • isEmpty() — проверка пустоты O(1)

🚶 Очередь (Queue) — FIFO

First In, First Out — первый вошёл, первый вышел. Как очередь в магазине.

Основные операции:
  • enqueue(x) — добавить в конец O(1)
  • dequeue() — удалить из начала O(1)
  • front() — посмотреть первый элемент O(1)
  • isEmpty() — проверка пустоты O(1)

Виды очередей

Deque (Double-Ended Queue)
Добавление и удаление с обоих концов O(1). Python collections.deque.
Priority Queue
Извлечение по приоритету, а не по порядку добавления. Реализуется через Heap.
Circular Queue (кольцевая)
Эффективное использование памяти: конец соединяется с началом. Реализуется на массиве.
Monotonic Queue
Очередь с сохранением монотонности (возрастания/убывания). Скользящее окно.

Где используются?

  • Стек: Отмена действий (Undo), история браузера, рекурсия (call stack), парсинг выражений
  • Очередь: Очередь печати, обработка задач (task queue), BFS
  • Приоритетная очередь: Планировщик ОС, Dijkstra, сжатие Хаффмана
  • Deque: Sliding Window, LRU Cache
