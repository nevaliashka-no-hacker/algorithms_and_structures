🔗 Связные списки (Linked Lists)

Что такое связный список?

Связный список — это линейная динамическая структура данных, где элементы (узлы) хранятся в разных участках памяти и связаны указателями. В отличие от массива, не требует непрерывного блока памяти.

Ключевые термины:
  • Узел (Node) — содержит данные и указатель(и) на следующий(е) узел(ы)
  • Голова (Head) — первый узел списка
  • Хвост (Tail) — последний узел (указатель на NULL)
  • NULL / nullptr — маркер конца списка

Виды связных списков

Односвязный (Singly Linked)
Каждый узел → один указатель на следующий. Однонаправленный обход.
Двусвязный (Doubly Linked)
Два указателя: на следующий и предыдущий. Двунаправленный обход.
Циклический (Circular)
Последний узел указывает на первый (кольцо). Нет NULL.

Массив vs Связный список

ОперацияМассивСвязный список
Доступ по индексуO(1)O(n)

Где используются связные списки?

  • 🧠 Управление памятью ОС: Списки свободных блоков
  • 🎵 Музыкальные плееры: Плейлисты (следующий/предыдущий трек)
  • 🖼️ Графические редакторы: История действий (Undo/Redo)
  • 📝 Текстовые редакторы: Буфер текста (вставка без сдвига)
  • ⛓️ Хеш-таблицы: Метод цепочек для коллизий
  • 🔄 LRU Cache: Двусвязный список + хеш-таблица
