💰 Жадные алгоритмы (10 алгоритмов)

🎯 Жадные — когда локально оптимальный выбор ведёт к глобально оптимальному решению.

Принцип работы

Жадный алгоритм на каждом шаге делает наилучший выбор в данный момент, не думая о будущем. Работает не всегда, но когда работает — очень быстро.

Доказательство корректности:
  • Greedy Choice Property: глобально оптимальное решение содержит локально жадный выбор
  • Optimal Substructure: после жадного выбора остаётся подзадача той же структуры
1. Activity Selection (Задача о выборе заявок) ⭐⭐⭐⭐

Даны интервалы [start, end]. Выбрать максимальное число непересекающихся. Сортируем по концу, берём совместимые.

ВремяO(n log n)ПамятьO(1)
2. Fractional Knapsack (Дробный рюкзак) ⭐⭐⭐

Можно брать часть предмета. Сортируем по удельной стоимости (цена/вес), берём максимум.

3. Код Хаффмана ⭐⭐⭐⭐

Оптимальное префиксное кодирование. Строим дерево снизу, объединяя два самых редких символа.

ВремяO(n log n)
4. Размен монет (жадный) ⭐⭐⭐

Для канонических систем (1, 2, 5, 10) работает жадный. Для произвольных — только DP.

5. Meeting Rooms (конференц-залы) ⭐⭐⭐⭐

Сколько залов нужно для всех встреч? Сортируем по началу, приоритетная очередь по концу.

6. Задача о расписании (Job Sequencing) ⭐⭐⭐

Работы с дедлайнами и прибылью. Максимизировать прибыль. Сортируем по прибыли, ставим ближе к дедлайну.

7. Минимальное число платформ (Train Platform) ⭐⭐

Даны времена прибытия и отправления поездов. Сколько платформ нужно? Сортируем оба массива, два указателя.

8. Jump Game (игра в прыжки) ⭐⭐⭐

Можно ли допрыгать до конца? Greedy: поддерживаем furthest достижимый индекс.

ВремяO(n)
9. Gas Station (заправки) ⭐⭐⭐

Где начать, чтобы объехать круг? Greedy: если сумма gas-cost ≥ 0, решение есть. Ищем старт.

10. Разбиение строки (Partition Labels) ⭐⭐

Разбить строку на максимум частей, где каждый символ только в одной части. Храним последнее вхождение.
