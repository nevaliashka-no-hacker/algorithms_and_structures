🔍 Бинарный поиск (12 паттернов)

🎯 Бинарный поиск — не только поиск в массиве. Это целый класс алгоритмов поиска по ответу!

Классический бинарный поиск ⭐⭐⭐⭐⭐

Делим отсортированный массив пополам, отсекаем половину. O(log n).

Ключевые ошибки новичков:
  • Не mid = (l + r) // 2, а mid = l + (r - l) // 2 (защита от переполнения в C/C++)
  • Путаница с < vs <= в условии while
  • Выбор mid / mid - 1 / mid + 1 при сужении границ

📌 Паттерны бинарного поиска

1. Lower Bound (первый ≥ x) ⭐⭐⭐⭐⭐

Найти первый элемент, который ≥ x. bisect_left в Python.

2. Upper Bound (первый > x) ⭐⭐⭐⭐⭐

Найти первый элемент > x. bisect_right в Python.

3. Поиск в Rotated Sorted Array ⭐⭐⭐⭐

Массив сдвинут (4,5,6,7,0,1,2). Определяем, в какой половине искать.

4. Поиск пика (Peak Element) ⭐⭐⭐⭐

Найти элемент, который ≥ соседей. Смотрим на mid и его соседа, идём в сторону возрастания.

5. Поиск в 2D-матрице ⭐⭐⭐

Каждая строка и столбец отсортированы. Начинаем с правого верхнего угла.

6. Бинарный поиск по ответу (на минимум/максимум) ⭐⭐⭐⭐

Угадываем ответ и проверяем функцией check(). Делим диапазон возможных ответов.

7. Поиск минимального в Rotated Sorted Array ⭐⭐⭐⭐

Найти точку поворота. Сравниваем mid с правым концом.

8. Поиск пропущенного числа

В отсортированном массиве 0..n пропущено одно. Сравниваем arr[mid] с mid.

9. Koko Eating Bananas (пример поиска по ответу) ⭐⭐⭐

Минимальная скорость поедания. guess = скорость, check = успеет ли за H часов.

10. Split Array Largest Sum ⭐⭐⭐

Разбить массив на k частей, минимизировать максимальную сумму части. Поиск по ответу.

11. Экспоненциальный поиск (Exponential Search)

Для неограниченных массивов. Удваиваем индекс, затем бинарный поиск. O(log n).

12. Троичный поиск (Ternary Search) ⭐⭐

Для унимодальных функций (один пик). Делим на три части. O(log₃ n).
