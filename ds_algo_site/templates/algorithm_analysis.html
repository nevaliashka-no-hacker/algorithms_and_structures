Master Theorem (Основная теорема) ⭐⭐⭐⭐⭐

Для рекуррент: T(n) = a * T(n/b) + f(n), где a ≥ 1, b > 1.

Сравниваем f(n) с n^(log_b a):

СлучайУсловиеРешение
1f(n) = O(n^(log_b a - ε))T(n) = Θ(n^(log_b a))
2f(n) = Θ(n^(log_b a))T(n) = Θ(n^(log_b a) * log n)
3f(n) = Ω(n^(log_b a + ε)) + условие регулярностиT(n) = Θ(f(n))
Примеры:
  • Merge Sort: T(n)=2T(n/2)+O(n) → a=2,b=2,log₂2=1, f(n)=O(n) → Случай 2 → Θ(n log n)
  • Бинарный поиск: T(n)=T(n/2)+O(1) → a=1,b=2,log₂1=0, f(n)=O(1) → Случай 2 → Θ(log n)
  • DFS дерева: T(n)=2T(n/2)+O(1) → Случай 1 → Θ(n)

Амортизационный анализ ⭐⭐⭐⭐

Средняя стоимость операции в худшей последовательности операций.

  • Aggregate Method: суммируем все операции, делим на их количество
  • Banker's Method: "копим кредиты" на дешёвых операциях для дорогих
  • Potential Method: функция потенциала Φ (как высота подъёма воды)
Примеры:
  • Динамический массив: вставка O(1) амортизированное (удвоение размера)
  • DSU с эвристиками: почти O(1) (обратная функция Аккермана)
  • Хеш-таблица с рехэшированием: O(1) амортизированное
